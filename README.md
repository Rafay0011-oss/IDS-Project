# Smart City Management System

## Objective

The Smart City Management System is a database project designed to efficiently manage and organize information related to various city services. The system demonstrates the use of a relational database to store, retrieve, update, and manage data for smart city operations.

## Features

* Relational database design
* Data storage and management
* Primary and foreign key relationships
* SQL queries for data insertion, updating, deletion, and retrieval
* Efficient organization of city-related information

## Technologies Used

* SQL
* MySQL

## Project Files

* Database creation script
* Table creation script
* Data insertion/query scripts

## How to Run

1. Open MySQL Workbench (or any compatible SQL database management system).
2. Execute the SQL files in the correct order.
3. Create the database and tables.
4. Insert the required data.
5. Run the SQL queries to test the database.
